# AI Loan Approval Prediction

## Overview
This project develops a machine learning system to predict whether a loan application should be approved based on applicant information such as income, loan amount, credit history, employment status, and demographics.

## Business Problem
Financial institutions process large numbers of loan applications. Manual evaluation is time-consuming and may lead to inconsistent decisions. This project helps automate loan approval decisions while reducing risk.

## Objectives
- Automate loan eligibility assessment
- Reduce loan default risk
- Improve decision-making speed and consistency
- Support data-driven lending

## Dataset
- Records: 614
- Features: 13
- Target Variable: Loan_Status
